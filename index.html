Joining our Server

Java Edition

  1. Open Minecraft and Click "Multiplayer"
  2. Click "Add Server" at the bottom
  3. Insert the server address play.spookymc.xyz into the Server Address bar, and optionally change the server name to anything you wish. We recommended naming it SpookyMC.
  4. Click "Done" and Double Click our Server on the Server List.

Bedrock Edition

  1. Open Minecraft and Click "Play" and Click on "Servers"
  2. Scroll to the Bottom and Click "Add Server"
  3. Insert the server address pe.spookymc.xyz into the Server Address bar, 27842 as the Port, and optionally change the server name to anything you wish. We recommended naming it SpookyMC.
  4. Click "Play"
